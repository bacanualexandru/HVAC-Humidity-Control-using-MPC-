# HVAC-Humidity-Control-using-MPC
- Modeled a coupled HVAC humidity control system for interconnected greenhouse environments.
- Designed a constrained Model Predictive Control framework for humidity tracking under input and state constraints.
- Implemented and compared CVXPY, SciPy SLSQP, log-barrier gradient descent, and log-barrier Newton solvers.
- Evaluated solver performance using tracking MSE, control effort, and convergence behavior.
